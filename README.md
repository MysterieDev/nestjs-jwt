## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript repository for Authentication and Authorization

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Change the Database

dont put your DB-data in ormconfig.json publicly - deactivated revision using
git update-index --skip-worktree ormconfig.json
if you want to reactivate it use
git update-index --no-skip-worktree ormconfig.json

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
